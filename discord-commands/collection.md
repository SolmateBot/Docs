# /collection

Get detailed information about an NFT collection from Magic Eden.

## Usage

`/collection <name>`

| Option | Required | Description                                      |
| ------ | :------: | ------------------------------------------------ |
| `name` |    x     | Collection name, symbol, or Magic Eden link      |

## What it shows

* Collection name and image
* Floor price (SOL and USD)
* 24-hour volume and average price
* Total volume
* Number of listed NFTs
* Creation date
* Social media links
* Categories
